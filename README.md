# Shivam_website_

Shivam_website_ is a comprehensive landing page designed for a digital services agency specializing in SEO, Website Design, Development, Website Management, Social Media Management, and other digital services. The goal is to showcase the agency's offerings, highlight its expertise, and facilitate client engagement through a dedicated contact form.

## Overview

The application is a Single-Page Application (SPA) built using Vue.js. It's structured into components representing different sections of the landing page, including Hero Section, Services Overview, About Us, Contact Form, and Footer. The contact form integrates with Zoho CRM for efficient inquiry management. The site is deployed on Vercel, ensuring high availability and performance.

## Features

- **Responsive Landing Page:** Sections detailing services, agency information, and a contact form.
- **Zoho CRM Integration:** Direct submission of inquiries into Zoho CRM via the contact form.
- **Vue.js Components:** Modular architecture using Vue.js for dynamic user interactions.

## Getting started

### Requirements

- Node.js
- Vue CLI
- A modern web browser

### Quickstart

1. Clone the repository to your local machine.
2. Navigate to the project directory and install dependencies with `npm install`.
3. Start the development server with `npm run dev`.
4. Open your web browser and go to `http://localhost:3000` to view the application.
